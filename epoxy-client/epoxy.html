This is a proof of concept application to show:

  • that you don't need a web server
  • the web server framework can be a waste of time.
  • HTML5 changes the way things works with websockets
Of course you need a web server at customer installations. But as a developer you don't need one at all.

With proper css styling and some smart scripting you should be able to template everything with fever jscript lines than ever done with c++.

Server side code is simply processed by generating json stuff back and forth.
